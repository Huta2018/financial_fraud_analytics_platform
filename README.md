# Credit union fraud detection system:

This folder contains a machine learning project designed to detect suspicious financial transactions and risky payment-related communications. This focus on fraud analytics workflows relevant to credit unions and financial institutions.
# Objectives
- Detect fraudulent transactions using structured data
- Detect suspicious payment communications using text features
- Generate business-friendly fraud risk scores
- Provide interpretable model outputs for investigators
- Build an interactive Streamlit dashboard

## Tech Stack
Python, pandas, scikit-learn, SHAP, Streamlit, Plotly

## Planned Modules
1. Transaction fraud detection
2. Communication fraud detection
3. Risk scoring
4. Explainability
5. Dashboard
